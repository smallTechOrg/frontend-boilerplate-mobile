# Frontend Boilerplate Mobile

## Purpose
To be a starter repo to build a new mobile app from scratch, but with basic utilities included.


## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```


## Creating Boilerplate on your own
```
npx create-expo-app@latest project-folder
npm run reset-project
```
